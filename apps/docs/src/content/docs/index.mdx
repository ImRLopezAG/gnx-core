## What is GNX?

GNX is a collect of utilities that are used across all of our projects. It is a collection of utilities that we have found useful in our projects and we hope you will find them useful in yours. We have tried to make them as generic as possible so that they can be used in any project.

To install GNX, run the following command:

```sh
npm install @gnx-utilities/core
```

this will install the core package. If you want to install the other packages, you can do so by running the following command:

```sh
npm install @gnx-utilities/models @gnx-utilities/controllers @gnx-utilities/services @gnx-utilities/decorators
```
